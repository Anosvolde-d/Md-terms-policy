# Mdkey Bot - Policy Pages

This folder contains the policy pages for Mdkey Bot.

## Files

- `index.html` - Main landing page with links to all policies
- `privacy-policy.html` - Complete Privacy Policy
- `terms-of-service.html` - Complete Terms of Service
- `usage-guidelines.html` - Usage Guidelines

## How to Publish to GitHub Pages

### Option 1: Create a New Repository

1. Go to https://github.com/new
2. Name your repository (e.g., `mdkey-bot-policies`)
3. Make it **Public**
4. Click "Create repository"
5. Upload all HTML files from this folder
6. Go to repository Settings → Pages
7. Under "Source", select "main" branch
8. Click "Save"
9. Your site will be published at: `https://yourusername.github.io/mdkey-bot-policies/`

### Option 2: Use Existing Repository

1. Create a `docs` folder in your existing repository
2. Copy all HTML files to the `docs` folder
3. Go to repository Settings → Pages
4. Under "Source", select "main" branch and `/docs` folder
5. Click "Save"

## URLs for Discord Developer Portal

Once published, use these URLs in your Discord Application settings:

- **Privacy Policy URL**: `https://yourusername.github.io/mdkey-bot-policies/privacy-policy.html`
- **Terms of Service URL**: `https://yourusername.github.io/mdkey-bot-policies/terms-of-service.html`

Replace `yourusername` and `mdkey-bot-policies` with your actual GitHub username and repository name.
