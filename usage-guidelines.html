Usage Guidelines

Best practices for using Mdkey Bot

Getting Started

Mdkey Bot helps you configure your Discord server through natural conversation. Here's how to use it effectively:

Basic Commands

  • /setup - Start a conversational setup session
  • /config - Manage trusted roles and bot settings
  • /dryrun - Toggle preview mode without executing changes

Best Practices

1. Use Dry-Run Mode First

Before making changes to your server, enable dry-run mode to preview what the bot will do without actually executing the actions.

2. Be Specific in Your Requests

The more specific you are about what you want, the better the bot can help you. Instead of "set up channels," try "create a welcome category with rules and announcements channels."

3. Review Before Confirming

Always carefully review the action preview before confirming. The bot will show you exactly what it plans to do.

4. Configure Trusted Roles

Set up trusted roles to control who can use the bot's setup commands. Only administrators and trusted roles can execute server changes.

5. Backup Important Data

Before making significant changes to your server, consider backing up important information or taking screenshots of your current setup.

What You Can Do

  • Create and organize channels and categories
  • Configure channel permissions for roles and users
  • Set up read-only channels and moderator-only areas
  • Post messages and templates to channels
  • Create threads in existing channels
  • Modify channel properties (topic, slowmode, NSFW)

What to Avoid

  • Don't use the bot to spam or flood your server
  • Don't attempt to exploit or abuse the bot's functionality
  • Don't use the bot for malicious purposes
  • Don't share your bot configuration with untrusted users
  • Don't make destructive changes without careful consideration

Rate Limits

To prevent abuse, the bot implements rate limiting:

  • 10 requests per minute per user
  • 100 requests per hour per user

If you hit the rate limit, wait a few minutes before trying again.

Getting Help

If you need assistance or have questions:

  • Review the bot's responses carefully - they often contain helpful guidance
  • Use dry-run mode to experiment safely
  • Contact us at Mdkey.Bot@gmail.com for support

Example Conversations

Creating a Welcome Section

You: /setup

Bot: Hello! What would you like to configure?

You: Create a welcome section with rules and announcements

Bot: I'll create a "Welcome" category with two channels... Would you like me to proceed?

You: yes

Bot: Successfully completed all actions!

Tips for Success

  • Start with small changes to get familiar with the bot
  • Use natural language - the bot understands conversational requests
  • Ask the bot for suggestions if you're unsure what to do
  • Take advantage of the confirmation step to review changes
  • Keep your server organized by using categories effectively
